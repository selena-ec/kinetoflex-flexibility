# Intermediate Flexibility Rolling Plan

## Summary

This is a 4-week Intermediate-level flexibility plan based on the four areas in The Ultimate Flexibility Bundle:

- Hip Mobility
- Front Split
- Middle Split
- Pancake

The plan is organized by week, Sunday through Saturday. Each week includes 8 flexibility exercises. Train daily with one exercise per day, except Friday, which has two exercises.

Follow-along videos count as alternate formats for the same workout. They are not separate sessions.

## Intermediate Workouts Included

Use only these 8 sessions:

- Hip Mobility Intermediate - Workout 1
- Hip Mobility Intermediate - Workout 2
- Front Split Intermediate - Workout 1
- Front Split Intermediate - Workout 2
- Middle Split Intermediate - Workout 1
- Middle Split Intermediate - Workout 2
- Pancake Intermediate - Workout 1
- Pancake Intermediate - Workout 2

## Weekly Layout

| Day | Sessions |
|---|---|
| Sun | Hip Mobility Intermediate - Workout 1 |
| Mon | Front Split Intermediate - Workout 1 |
| Tue | Middle Split Intermediate - Workout 1 |
| Wed | Pancake Intermediate - Workout 1 |
| Thu | Hip Mobility Intermediate - Workout 2 |
| Fri | Front Split Intermediate - Workout 2; Middle Split Intermediate - Workout 2 |
| Sat | Pancake Intermediate - Workout 2 |

## 4-Week Volume

The tracker covers 4 weeks. Each week has 8 exercises, for 32 total tracked flexibility exercises.

| Area | Sessions Per 4 Weeks |
|---|---:|
| Hip Mobility | 8 |
| Front Split | 8 |
| Middle Split | 8 |
| Pancake | 8 |

Each area includes 4 sessions of Workout 1 and 4 sessions of Workout 2.

## Week Notes

At the end of each week, briefly record:

- Energy level
- Soreness level
- Range or comfort changes
- Anything pinchy, sharp, or painful
- Which area felt best
- Which area felt most restricted

## Progress Rule

Stay at Intermediate for the full 4-week plan unless the workouts feel controlled, pain-free, and repeatable. If you need more time at this level, repeat the 4-week plan before changing levels again.
