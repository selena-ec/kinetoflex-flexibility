# Ballet Kinetoflex Flexibility Tracker

Beginner flexibility tracker for an 8-week Kinetoflex ballet support plan.

## Web App

Open the live app here:

https://selena-ec.github.io/ballet-kinetoflex-flexibility/

## Plan

- 8 weeks
- 64 tracked flexibility exercises
- 8 weekly Sun-Sat plans
- 4 flexibility workout days per week
- 3 open days per week for other training
- 4 focus areas: Hip Mobility, Front Split, Middle Split, Pancake
- Google Sheets sync support through Apps Script

## Files

- `index.html` - app shell
- `styles.css` - app styling
- `app.js` - tracker logic and sync behavior
- `config.js` - Google Apps Script endpoint configuration
- `google-apps-script.gs` - Google Sheets backend
- `beginner-flexibility-rolling-plan.md` - plan details
- `google-sheets-backend-setup.md` - backend setup notes
